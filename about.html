Über das Projekt

Eine kleine 1-Pixel-Kamera misst die Farbe des Himmels über Karlsruhe und überträgt die Werte live auf die Webseite. Hier kann man immer sehen, wie der Himmel über Karlsruhe gerade ist.

Warum?

Wenn wir etwas teilen, wenn wir am gleichen Ort sind, dann ist es Licht. Gleiche Sonne, gleicher Himmel, gleiches Licht.
Licht erzeugt einen gemeinsamen Raum. Licht schafft Verbundenheit, ob am Lagerfeuer, bei Kerzenschein oder über der Stadt. Dieses Projekt ist eine Erinnerung an den Raum den wir teilen.

Lichtarchiv

Über Zeit sammeln sich die Lichtdaten und erzählen eindrucksvoll von Gewittern, von bewölkten Tagen und von sonnigen, von frühen rosigen Sonnenaufgängen, von langen Sommertagen und kurzen Wintertagen.
Für das Archiv wird nicht nur ein einziger Lichtwert gespeichert, sondern ein ein-pixel-breiter Ausschnitt des Himmels in Süd-Nord-Richtung. Bei den entstandenen Bildern ist somit die X-Achse die Zeitachse, und die Y-Achse die Nord-Süd-Achse des Himmels.

Himmel über Karlsruhe ist ein Projekt von Michail Rybakov und Petko Godev.

API

Der API-Endpoint für den aktuellen Licht-Farb-Wert ist hier verfügbar: https://api.himmelueberkarlsruhe.de/last Man kann die API nutzen, um z.B. die aktuelle Farbe in das Design einer Webseite zu integrieren, oder auch (mit etwas mehr Arbeit) um eine IoT-Lampe mit Karlsruher Licht leuchten zu lassen.

Code

Der Code für das Projekt ist auf Github verfügbar: API, Webseite und Raspberry Pi.

Himmel über Karlsruhe wird unterstützt durch die UNESCO City of Media Arts Karlsruhe und die Stadt Karlsruhe.
